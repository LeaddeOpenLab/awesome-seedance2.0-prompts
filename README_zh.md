## 🤔 什么是 Seedance 2.0？

**Seedance 2.0** 是**字节跳动**推出的视频生成模型，支持文本、图片、视频和音频输入，适合电影化短片、产品展示、社媒内容和多模态视频实验。

**Key Features:**
- 🎥 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片动画化为动态视频
- 📹 **视频转视频** — 转换和扩展现有视频
- 🎵 **音频驱动** — 通过音频输入驱动视频生成
- 📐 **最高 1080p 分辨率**，4-15 秒时长
- 🔊 **自动配音配乐** — 自动旁白和背景音乐
